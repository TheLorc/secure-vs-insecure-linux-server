# Secure Ubuntu Server (VM)

## Overview
This VM was hardened from an insecure state using basic Linux security best practices.

## Improvements Made
- Disabled SSH root login
- Enabled SSH key authentication
- Configured UFW firewall
- Removed unnecessary services
- Updated system packages
- Improved file permissions
- Installed Fail2Ban for brute-force protection

## Result
The server is now more secure with reduced attack surface and improved access control.
